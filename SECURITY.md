# Security Policy

Security issues in `otta-cli` are handled with priority.

## Supported Versions

Use the latest version from this repository and GitHub releases.
Older snapshots may not receive security fixes.

## Reporting a Vulnerability

If you believe you found a vulnerability, report it privately through
GitHub Security Advisories for this repository.

Please include:

- Reproduction steps
- Affected command(s)
- Impact assessment
- Any proof-of-concept payloads (if safe to share)

## Disclosure

Please avoid public disclosure before coordinated remediation.
